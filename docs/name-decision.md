# 插件名称确认

当前名称：`Web QA Inspector`

## 调研结论

截至 2026-04-25，公开搜索未发现以下商店存在明显精确同名冲突：

- Chrome Web Store
- Firefox Add-ons
- Microsoft Edge Add-ons

搜索过的关键词包括：

- `Web QA Inspector`
- `Web QA Inspector Chrome Web Store`
- `Web QA Inspector addons.mozilla.org`
- `Web QA Inspector Microsoft Edge Add-ons`

## 风险说明

`Web`, `QA`, `Inspector` 都是描述性通用词，因此不能视为强品牌名。商店最终是否允许使用该名称，以提交审核时的结果为准。

## 当前建议

继续使用 `Web QA Inspector` 作为第一版名称，原因：

- 名称直观说明插件用途。
- 已经与当前图标、商店图、文档和隐私政策一致。
- 暂未发现精确同名冲突。
- 第一版更需要清晰表达功能，而不是过早建立抽象品牌。

## 备选名称

如果后续商店审核或品牌策略要求改名，建议选择更独特的名称：

1. `LaunchLens QA`
2. `PageLint Studio`
3. `SiteProof QA`
4. `Preflight Web QA`
5. `PageCheck Inspector`

改名时必须同步更新：

- manifest `_locales` 名称。
- Popup / Options 标题。
- 隐私政策。
- 服务条款。
- 商店文案。
- 商店图。
- GitHub Pages 页面标题。
- Release zip 名称可选同步。
