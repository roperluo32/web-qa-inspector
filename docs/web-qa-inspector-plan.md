# Web QA Inspector 免费版开发计划

当前版本定位为完全免费的本地网页质量检查插件。

## 免费功能

- 当前页面本地扫描。
- SEO 基础检查。
- Open Graph / Twitter Card 检查。
- 链接检查。
- 图片检查。
- 可访问性基础检查。
- JSON-LD 结构化数据检查。
- 本地扫描历史。
- HTML / CSV 报告导出。
- 多语言 UI 与多语言商店素材。

## 明确不包含

- 不包含账号系统。
- 不包含远程授权码。
- 不包含远程同步。
- 不默认上传页面内容。

## 测试要求

- TypeScript typecheck 通过。
- ESLint 通过。
- Vitest 单元测试通过。
- Playwright E2E 测试通过。
- Chrome / Edge / Firefox 构建通过。
