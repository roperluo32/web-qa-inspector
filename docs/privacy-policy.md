# Web QA Inspector 隐私政策草稿

最后更新：2026-04-25

Web QA Inspector 是一个本地网页质量检查浏览器插件，用于检查当前网页的 SEO、链接、图片、可访问性基础项和结构化数据。

## 数据处理原则

- 插件默认在本地浏览器中扫描当前页面。
- 插件不会默认上传网页正文、DOM、截图或扫描结果到服务器。
- 扫描历史保存在浏览器本地 storage 中，用户可以在 Options 页面清除。
- 免费功能不需要账号登录。

## 插件会访问的数据

为了完成用户主动触发的扫描，插件可能读取当前页面的以下信息：

- 页面 title、meta 标签、canonical、lang、viewport。
- 页面链接 href。
- 图片 src、alt、width、height、loading 属性。
- 标题、按钮、表单控件等基础可访问性相关 DOM 信息。
- JSON-LD 结构化数据脚本内容。

这些数据用于生成本地扫描结果。

## 本地保存的数据

插件会在浏览器本地保存：

- 最近扫描历史。
- 用户选择的界面语言。

## 第三方服务

当前版本为免费插件，不需要账号登录，也不会保存账号、付款或远程授权信息。

## 权限说明

- `activeTab`：用户主动点击扫描时访问当前标签页。
- `scripting`：向当前网页注入一次性扫描脚本。
- `storage`：保存本地历史和界面语言设置。
- `tabs`：获取当前标签页 ID、URL 和标题，用于扫描和历史记录。
- `http://*/*`、`https://*/*`：允许在普通网页中执行用户主动触发的扫描。插件不会后台主动扫描所有网页。
