# Web QA Inspector 商店文案草稿

## 短描述

Local SEO, link, image, accessibility, and structured data audits for any web page.

## 长描述

Web QA Inspector helps developers, SEO consultants, content teams, and agencies catch common page quality issues before launch.

Run a local audit from the browser popup and review issues across:

- SEO basics: title, meta description, H1, canonical, lang, viewport, robots.
- Social previews: Open Graph and Twitter Card metadata.
- Links: empty hrefs, JavaScript links, hash-only links, duplicate links, external links.
- Images: missing alt text, missing src, dimensions, lazy loading hints.
- Accessibility basics: heading order, button names, form labels.
- Structured data: JSON-LD presence, parse validity, and @type hints.

Privacy-first by default:

- Scans run locally in your browser.
- Page content is not uploaded by default.
- Scan history is stored locally and can be cleared.

## 权限用途说明

- activeTab: access the current tab after the user starts a scan.
- scripting: run the local scanner on the current page.
- storage: save local scan history and settings.
- tabs: identify the active tab and display URL/title information.
- http/https host access: allow scans on ordinary web pages when triggered by the user.
